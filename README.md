# Interactive Transcript Quizzer 🧠

A custom-built, responsive web application designed to enhance learning retention through active recall. This project takes YouTube lecture transcripts for complex subjects and uses Large Language Models (LLMs) to automatically generate comprehensive, interactive quizzes. 

Currently, this repository hosts study materials for subjects like **DBMS**, **Natural Language Processing (NLP)**, and **Deep Learning** (aligned with the IITM-BS curriculum structure).

## ✨ Features

* **AI-Generated Content:** Questions, contexts, and detailed explanations are generated by feeding YouTube lecture transcripts into LLMs, ensuring the quizzes are perfectly aligned with the course material.
* **Keyboard Shortcuts:** fully mapped keyboard shortcuts for power users to navigate questions, select options, and check answers without needing a mouse.
* **Responsive Design:** A mobile-friendly UI that scales perfectly whether you are studying on a desktop, tablet, or smartphone.
* **Multiple Question Types:** Supports standard Multiple Choice Questions (MCQs) and Type-in/Fill-in-the-blank questions.
* **Context-Aware Explanations:** Every question provides the specific lecture context before answering, and a detailed explanation afterward to reinforce the concept.

## 🛠️ Data Structure

The application reads quiz data from structured JSON files. This makes it incredibly easy to scale and add new lectures. Here is an example of the schema used for the quizzes:

### MCQ Format

```json
{
  "id": 1,
  "type": "mcq",
  "context": "The NLP pipeline requires raw text to be cleaned...",
  "question": "What is the process of breaking text into smaller pieces called?",
  "options": [
    "A) Normalization",
    "B) Stop word removal",
    "C) Tokenization",
    "D) Lemmatization"
  ],
  "answer": "C) Tokenization",
  "explanation": "Tokenization is the first step of 'reading' for a computer..."
}

```

### Type-in Format

```json
{
  "id": 4,
  "type": "type-in",
  "context": "We want to use NLTK's sentence tokenizer...",
  "question": "Import the sentence tokenizer function from the tokenize module...",
  "answer": "from nltk.tokenize import sent_tokenize",
  "explanation": "This imports the sent_tokenize function..."
}

```

## 🚀 Usage

1. Clone this repository to your local machine.
2. Open the root `index.html` file in any modern web browser to access the main dashboard.
3. Navigate to your desired subject (e.g., `nlp` -> `week-2` -> `lec_2.1.html`).
4. Start testing your knowledge!

## 💡 How to Add New Quizzes

1. Copy the YouTube transcript of your lecture.
2. Prompt an LLM to generate a JSON array matching the structure shown above.
3. Save the output as a `.json` file in the appropriate `static/js/week-X/` directory.
4. Link the new JSON file to your frontend logic.
